# Telegram Social Downloader Bot

## Features

- TikTok No Watermark
- Facebook Downloader
- Instagram Downloader
- YouTube Downloader
- SoundCloud Downloader
- Auto Detect URL
- Auto Cleanup
- Telegram Upload

---

# Install

```bash
pip install -r requirements.txt
```

---

# Run

```bash
python bot.py
```

---

# Environment Variable

```bash
BOT_TOKEN=YOUR_BOT_TOKEN
```
